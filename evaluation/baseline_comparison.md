# Lightweight Baseline Comparison

## Purpose

This evaluation compares the improved Weather Insight App against a simple baseline weather response.

The baseline only provides basic weather information, while the improved app provides practical recommendations and risk-aware advice based on weather conditions.

## Baseline System

The baseline system returns only a simple weather summary, such as:

- City name
- Temperature
- Weather condition
- Humidity
- Wind speed

Example baseline response:

> The weather in Toronto is 12°C with cloudy conditions, 70% humidity, and wind speed of 15 km/h.

## Improved App

The improved Weather Insight App gives the user more useful guidance by adding:

- A clear weather summary
- Practical activity recommendations
- Risk-aware advice for rain, cold, heat, humidity, or wind
- Better handling of invalid city names

Example improved response:

> The weather in Toronto is 12°C with cloudy conditions, 70% humidity, and wind speed of 15 km/h. Wear a light jacket and be cautious of wind if walking or biking outside.

## Comparison

| Criteria | Baseline Weather Response | Improved Weather Insight App |
|---|---|---|
| Weather summary | Yes | Yes |
| Practical recommendation | No | Yes |
| Risk-aware advice | No | Yes |
| Invalid city handling | Basic or unclear | Clear message asking for a valid city |
| User value | Low | Higher |

## Conclusion

The improved Weather Insight App performs better than the baseline because it does not only return raw weather information. It also gives practical advice that helps users make better daily decisions, such as what to wear, whether to carry an umbrella, or whether to avoid outdoor activity during risky weather conditions.
